# awesome-ocr-collection

项目起源：发现现有的awesome-ocr系列都不怎么更新了，但是相关OCR工作却一直出来。为了便于自己时刻跟踪前沿，想着将已有的项目做一个融合，同时添加新的工作进来。

思路是采用双轨：
- 学术工作
- 工程项目
